# 🐳 Docker Learning Project #1 – Flask Quiz Application

This repository isn't just a Flask application—it's a milestone in my Docker learning journey.

Instead of learning Docker only through tutorials, I wanted to understand it by building and containerizing a real application.

## 💡 What I Built

A simple quiz application where users can:

* Answer multiple-choice questions
* Submit their responses
* View their final score instantly

The application is built with Flask and runs inside a Docker container.

## 🎯 Why I Built This

I wanted to understand:

* How Docker packages an application
* The difference between an Image and a Container
* How Docker Compose simplifies running applications
* How a Flask application can run consistently across different machines

This project helped me connect Docker concepts with a real application instead of just running Docker commands.

## 🧠 What I Learned

* Writing a Dockerfile
* Building Docker images
* Running containers
* Docker Compose
* Port mapping
* Project organization for Flask applications
* Basic Docker workflow

## 🚀 Next Goal

This is the first project in my Docker learning series.

Next, I'll containerize applications that include databases and multiple services to understand how Docker is used in real-world development.
